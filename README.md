# Parcial 2 - Tienda en Linea

## Algoritmos Implementados

### Ordenamiento (3 algoritmos)
1. **Merge Sort** - O(n log n)
2. **Quick Sort** - O(n log n) promedio, O(n²) peor caso
3. **Insertion Sort** - O(n²)

### Búsqueda (2 algoritmos)
1. **Búsqueda Binaria** - O(log n) (requiere datos ordenados por ID)
2. **Búsqueda Lineal** - O(n) (búsqueda por subcadena en nombre)
